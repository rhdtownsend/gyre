.. _poly-models:

Polytropic Models
=================

.. nml:group:: model
   :no-target:

Setting the :nml:option:`model_type` option of the :nml:group:`model`
namelist group to :nml:value:`'POLY'` tells the frontend to read the
equilibrium stellar model from a :ref:`POLY file <poly-file-format>`
created by the :program:`build_poly` tool. See the :ref:`comp-ptrope`
and :ref:`support-tools` appendices for further details.
